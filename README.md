# 泉流迹 Quellog

> 日用之流，静静有迹。

泉流迹（Quellog）是一款面向家庭与个人的开销记录和统计工具。
它关注日常消费的持续记录、清晰归类与长期回看，帮助使用者理解钱花在哪里、哪些支出正在变化，以及家庭或个人预算是否保持在可控范围内。

## 项目定位

日常开销往往像水流一样细碎、连续，却会在时间中留下清晰的轨迹。
泉流迹希望把这些轨迹安静地记录下来，让每一次消费都能被整理、被统计、被看见。

## 当前工程结构

- `firmware/`：墨水屏终端固件骨架，基于 ESP-IDF，默认目标为 `Zectrix S3 e-paper 4.2"`
- 服务端：待建设

本项目适合用于：

- 记录个人日常消费
- 管理家庭共同开销
- 按分类、账户、成员或时间维度统计支出
- 回顾月度、季度、年度消费趋势
- 辅助制定和调整家庭或个人预算

## 核心功能规划

- 开销记录：记录金额、分类、时间、备注等信息
- 收入记录：保留工资、报销、转账等收入流水
- 分类管理：支持自定义消费分类，适配不同家庭和个人习惯
- 统计分析：按日、月、年查看收支概览和趋势
- 家庭视角：支持多人或多成员维度的支出归属
- 数据筛选：按时间、分类、关键词快速查找记录
- 预算辅助：为重点分类或周期设置预算，并观察执行情况
